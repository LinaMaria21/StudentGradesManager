
Student Grad eManager
C# Console application
This is a console app to manage grades and to practice data structures and algorithms.

.NET 8

Features
Students can:

- Store grades in a List<int>
- Display all grades entered
- Calculate the average grade
- Demonstrates loops, lists and methods


How to run
1. Install .NET 8 SDK  
2. Clone this repository  
3. Open the project in Visual Studio or VS Code  
4. Run the program using dotnet run
